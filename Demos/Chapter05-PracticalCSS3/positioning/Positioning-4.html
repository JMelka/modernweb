Positioning - Basics

Absolute positioning of a child element within a container element. Here the container is positioned, i.e. we've set position:relative set. Therefore the positioned element is positioned relative to this container element. All browsers operate in the same way. 

<div style="position:relative; width:400px; height:400px; 
            margin:0; border:5px solid #369; padding:0; background:yellow;">
   Container Element
      <div style="position:absolute; left:50%; top:50%; width:50%; height:50%; 
                  margin:0; border:5px solid #f00; padding:0; background:pink; ">
         Positioned element
      </div>
</div>
